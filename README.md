# E2E Testing Pipeline

AI-powered end-to-end testing pipeline with CI/CD automation and real-time notifications.

## Features

- Automated CI/CD pipeline using GitHub Actions
- Mock server simulating real application behavior
- API testing with Jest
- AI-based validation using Google Gemini
- Telegram notifications for test results

## Workflow

1. Start mock server
2. Run automated API tests
3. Validate application using AI
4. Send results via Telegram

## Purpose

This project demonstrates how QA workflows can be automated using modern tools, combining traditional testing with AI-based validation.

## Notes

Production code is private, so a mock server is used to simulate real-world testing scenarios.
